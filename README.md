# tvu-shared-memory
1. This is a sub-project of MediaMesh that demonstrates how to interact with TVU's shared memory system using the provided SDK.
   You will find the binary inside `/lib/` from the tar.xz file in the **Release** folder of this repo.
   The `/test/` directory has the read/write sample code for reference.
